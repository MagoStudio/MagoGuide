# Diagrams

Most diagrams in these docs are inline [Mermaid](https://mermaid.js.org/) code blocks, which render natively on GitHub and stay editable in version control.

This folder holds **hand-tuned diagrams** that need layout Mermaid can't express, committed as `.svg` (with any editable source alongside). Embed them from the relevant page with a relative path.
