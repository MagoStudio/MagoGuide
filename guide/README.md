## The five workspaces

Mago's editor is divided into five workspaces, one per major task.

| Workspace | Use it to… |
| --- | --- |
| [Render Video](workspaces/render-video.md) | Generate video renders from a source video (the main workspace) |
| [Modify Frame](workspaces/modify-frame.md) | Stylize or edit a single frame before committing to video |
| [Mask](workspaces/mask.md) | Define which regions of a video get edited |
| [Upscale](workspaces/upscale.md) | Increase resolution and detail of a finished render |
| [Global Timeline](workspaces/global-timeline.md) | Review pinned renders across all shots in a project |
