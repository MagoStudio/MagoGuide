## The five workspaces

Mago's editor is divided into five workspaces, one per major task.

| Workspace | Use it to… |
| --- | --- |
| [Render Video](/guide/workspaces/render-video) | Generate video renders from a source video (the main workspace) |
| [Modify Frame](/guide/workspaces/modify-frame) | Stylize or edit a single frame before committing to video |
| [Mask](/guide/workspaces/mask) | Define which regions of a video get edited |
| [Upscale](/guide/workspaces/upscale) | Increase resolution and detail of a finished render |
| [Global Timeline](/guide/workspaces/global-timeline) | Review pinned renders across all shots in a project |
