# Keyboard Shortcuts

[← Quick reference](README.md)

---

| Shortcut | Action | Context |
| --- | --- | --- |
| **Space** | Play / pause | Viewport |
| **↑ / ↓** | Select previous / next track | Timeline |
| **← / →** | Move frame cursor one frame | Timeline |
| **Shift + S** | Show / hide Settings panel | Anywhere in editor |
| **Ctrl/Cmd + click** | Select additional track for comparison view | Timeline |
| **Ctrl + Shift + D** | Open the Debug Info panel | Anywhere in editor |
| **Double-click track name** | Rename track | Timeline |
| **Double-click project name** | Rename project | Top bar or project list |
